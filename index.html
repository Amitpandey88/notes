SketchNotes

A free, open-source collaborative whiteboard and note-taking app — right in your browser.

Draw with pen & shapes, write text, organize boards, export to PNG or SVG, and share with anyone via a link.

Open Whiteboard View Documentation GitHub Repo

Features

Everything you need to sketch, plan, and share ideas — no sign-up required.

🎨

Canvas Drawing

Pen, rectangles, ellipses, diamonds, lines, arrows, and text — all on an infinite canvas.

🔄

Undo / Redo

Full history with up to 50 steps so you can experiment freely.

📤

Export (PNG / SVG)

Download your boards as high-res PNG (2×, 4×, transparent) or crisp SVG.

🔗

Sharing

Generate read-only or read-write links to share boards instantly.

💾

Autosave

Auto-save to localStorage plus manual save to the server — never lose work.

🔍

Zoom & Pan

Mouse wheel, pinch-to-zoom, or the hand tool for smooth navigation.

📐

Grid Overlay

Toggle a grid for precise alignment and cleaner diagrams.

⌨️

Keyboard Shortcuts

Power-user shortcuts for every tool, undo, zoom, save, and more.

🆓

100% Free

No accounts, no limits, no tracking. Just open and create.

How It Works

Three simple steps — no setup, no sign-up.

1

Open

Launch the whiteboard in your browser — it works instantly.

2

Draw

Use the pen, shapes, text, and more to sketch your ideas.

3

Export or Share

Download as PNG/SVG or share a link with collaborators.

Installation

Set up the development environment in four commands.

# 1. Clone the repository
git clone https://github.com/Amitpandey88/notes.git
cd notes

# 2. Install dependencies
npm install

# 3. Run database migrations
npx prisma migrate dev --name init

# 4. Start the dev server
npm run dev

Then open http://localhost:3000 in your browser.
