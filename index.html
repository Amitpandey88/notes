SketchNotes

A free and open-source collaborative whiteboard and note-taking application. Draw, write, and organize your ideas — right in the browser.

star Features

  • 🎨 Canvas drawing with pen, shapes, lines, arrows, and text
  • 🔄 Undo/redo with full history (up to 50 steps)
  • 💾 Auto-save to localStorage and manual save to server
  • 📤 Export as PNG or SVG
  • 🔗 Share boards via read-only or read-write links
  • 🔍 Zoom & pan with mouse wheel, pinch, or hand tool
  • 📐 Optional grid overlay
  • 🗂 Board management — create, rename, duplicate, delete
  • ⌨️ Keyboard shortcuts for power users
  • 🆓 100% free — no accounts, no limits

get_app Installation

Get started in three easy steps:

# 1. Clone the repository
git clone https://github.com/Amitpandey88/notes.git
cd notes

# 2. Install dependencies
npm install

# 3. Set up the database and start the dev server
npx prisma migrate dev --name init
npm run dev

Then open http://localhost:3000 in your browser.
